Вызов функции

Для вызова функции пишут ее название и круглые скобки.

**Важно:** очень часто начинающие программисты забывают вызывать функцию. Помните, что объявление функции не вызывает ее.

```python
# объявление функции
def print_message():
    print('Я - Артур,')
    print('король британцев. ')


# вызов функции
print_message()
```
