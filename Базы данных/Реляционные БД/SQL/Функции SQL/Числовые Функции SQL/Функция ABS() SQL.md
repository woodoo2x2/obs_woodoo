Функция `ABS()` используется для вычисления модуля числа. Она принимает в качестве аргумента число и возвращает его модуль.

Результатом приведенного ниже запроса:

```python
SELECT ABS(-1),
       ABS(-2.1234),
       ABS(0),
       ABS(35);
```

является:

```no-highlight
+---------+--------------+--------+---------+
| ABS(-1) | ABS(-2.1234) | ABS(0) | ABS(35) |
+---------+--------------+--------+---------+
| 1       | 2.1234       | 0      | 35      |
+---------+--------------+--------+---------+
```
